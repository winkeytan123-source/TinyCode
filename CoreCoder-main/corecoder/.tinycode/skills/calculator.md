# 计算助手 Skill

## 触发条件
当用户提出以下类型的需求时，应主动加载并使用此 Skill：
- 数学计算（加减乘除、指数、对数、三角函数等）
- 公式推导与化简
- 单位换算（长度、重量、温度、进制等）
- 统计计算（均值、方差、标准差等）
- 金融计算（利息、复利、投资回报等）
- 任意需要精确数值结果的问题

## 工作流程

### 1. 理解需求
- 明确用户要计算什么
- 确认输入数值和单位
- 如果需求不清晰，先向用户确认

### 2. 选择计算方式
- **简单计算**：直接心算或笔算，给出结果
- **复杂计算**：使用 Python 代码精确计算，避免浮点误差
- **公式计算**：先写出公式，再代入数值

### 3. 执行计算
- 使用 `bash` 工具运行 Python 计算代码
- 涉及精度要求高的计算时，使用 `decimal` 模块
- 涉及符号运算时，使用 `sympy` 库（如已安装）

### 4. 输出结果
- 给出清晰的计算步骤
- 最终结果用 **粗体** 标出
- 如有多种表示方式，同时展示（如小数 + 分数）

### 5. 标记 Skill 使用
在输出结果的 **最末尾**，添加以下特殊标志行，表示本次回复使用了 calculator skill：

```
---
🧠 [Skill: calculator] 已触发
```

**规则：**
- 此标志必须出现在每次使用 calculator skill 后的输出末尾
- 标志行前后各空一行，使用 `---` 分隔线
- 不要修改或省略标志内容

## 计算模板

简单计算直接回答：
```
计算：3 + 5 × 2
步骤：先乘后除 → 5 × 2 = 10 → 3 + 10 = 13
结果：**13**

---
🧠 [Skill: calculator] 已触发
```

复杂计算使用 Python：
```python
import math
result = math.sqrt(2) ** 2
print(f"√2 的平方 = {result}")
```
输出后同样在末尾添加标志行。

## 注意事项
- 涉及大数计算时，使用 Python 而非心算，避免错误
- 单位换算要列出换算因子
- 金融计算要说明假设条件（如年利率、计息周期）
- 如果计算结果可能有多种解读，要逐一说明
